🧠 Как правильно внедрить тёмную тему

👉 у тебя должно быть 2 набора цветов


---

🎨 Светлая тема (у тебя уже есть)

:root {
  --bg: #F7F8FA;
  --card: #FFFFFF;
  --text: #111827;
  --secondary: #6B7280;
  --border: #E5E7EB;
  --primary: #4F46E5;
}


---

🌙 Тёмная тема (готовый вариант)

.dark {
  --bg: #0F172A;
  --card: #1E293B;
  --text: #F9FAFB;
  --secondary: #94A3B8;
  --border: #334155;
  --primary: #6366F1;
}


---

🧱 Использование в стилях

body {
  background: var(--bg);
  color: var(--text);
}

.card {
  background: var(--card);
  border: 1px solid var(--border);
}

---

⚙️ JS логика

function toggleTheme() {
  document.body.classList.toggle("dark")

  if (document.body.classList.contains("dark")) {
    localStorage.setItem("theme", "dark")
  } else {
    localStorage.setItem("theme", "light")
  }
}

// загрузка темы
if (localStorage.getItem("theme") === "dark") {
  document.body.classList.add("dark")
}
