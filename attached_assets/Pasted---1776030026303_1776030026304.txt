# 🧠 Суть (1 предложением)

👉 Бот получает сообщения → превращает их в заявку → отправляет на сервер → сайт показывает

---

# 🔗 Полная цепочка

```id="flow1"
Клиент → пишет боту  
↓  
Бот ведёт диалог  
↓  
Формирует заявку (JSON)  
↓  
Отправляет на backend  
↓  
Backend сохраняет в БД  
↓  
Сайт делает GET и показывает
```

---

# 🧱 Шаг 1. Бот сохраняет заявку

## 💡 Когда это происходит:

👉 когда диалог закончен
(есть: что, сколько, цена)

---

## 📦 Формируешь JSON:

```json id="json1"
{
  "business": "cake_moscow",
  "client_name": "Анна",
  "service": "Торт",
  "details": "Шоколадный, 2 кг",
  "price": 5000,
  "deadline": "2026-04-15",
  "status": "new"
}
```

---

## 🚀 Отправка на сервер:

```js id="code_send"
fetch("https://yourapi.com/leads", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify(data)
})
```

---

# 🧱 Шаг 2. Backend принимает

## 💡 Ты делаешь API:

```id="api1"
POST /leads
```

---

## Пример (Node.js / Express):

```js id="backend1"
app.post("/leads", (req, res) => {
  const lead = req.body

  // сохраняем в базу
  db.leads.insert(lead)

  res.json({ success: true })
})
```

---

# 🧱 Шаг 3. База данных

Минимум:

```js id="db1"
leads = [
  {
    id: 1,
    business: "cake_moscow",
    client_name: "Анна",
    price: 5000,
    status: "new"
  }
]
```

---

# 🧱 Шаг 4. Сайт получает заявки

## 💡 API:

```id="api2"
GET /leads?business=cake_moscow
```

---

## Backend:

```js id="backend2"
app.get("/leads", (req, res) => {
  const business = req.query.business

  const leads = db.leads.filter(
    l => l.business === business
  )

  res.json(leads)
})
```

---
